如何將 HTML 檔案轉換為 .exe (Windows)

HTML 檔案無法直接編譯，我们需要用 Python 做一個「外殼」來執行它。請依照以下步驟操作。

第一步：準備環境

安裝 Python (如果您尚未安裝)。

安裝必要的套件：
開啟命令提示字元 (CMD) 或 PowerShell，輸入以下指令安裝 pywebview (用來顯示網頁) 和 pyinstaller (用來打包)：

pip install pywebview pyinstaller


注意：pywebview 在 Windows 上需要 .NET 4.0 以上 (Windows 10/11 通常內建)。

第二步：準備檔案

請在桌面上建立一個新資料夾 (例如 MyWebApp)，並將以下 兩個檔案 放入其中：

您的 HTML 檔案 (例如 github_pushV2.html)。

啟動器腳本 (launcher.py) - 就是我剛剛產生的那個 Python 檔。

重要確認： 請打開 launcher.py，確認第 17 行的檔名是否與您的 HTML 檔名一致：

html_file = 'github_pushV2.html' 


第三步：執行打包指令

這是最關鍵的一步。我們需要告訴打包工具：「請把 HTML 檔案也一起塞進去」。

在 CMD 中，切換到該資料夾：

cd C:\Users\您的名字\Desktop\MyWebApp


執行打包指令 (Windows 專用語法)：
請完整複製以下指令並執行：

pyinstaller --noconsole --onefile --add-data "github_pushV2.html;." launcher.py


參數解釋：

--noconsole: 執行時不要跳出黑色視窗。

--onefile: 打包成單一 .exe 檔。

--add-data "github_pushV2.html;.": 這是最重要的一點！

格式為 "來源檔案;目標路徑" (Windows 使用分號 ; 分隔)。

這告訴程式把 github_pushV2.html 複製到 .exe 內部的根目錄 (.)。

launcher.py: 我們要打包的主程式。

如果您的 HTML 檔名不同，請記得修改指令中的檔名。

第四步：完成

等待打包完成後，進入新產生的 dist 資料夾。

您會看到 launcher.exe (您可以隨意重新命名它，例如 GitHubManager.exe)。

點擊執行，您應該會看到您的 HTML 介面在一個獨立的視窗中開啟，就像一個原生應用程式一樣！

常見問題 (HTML 特有)

1. 介面看起來怪怪的 / 樣式跑版？
由於您的 HTML (github_pushV2.html) 使用了 CDN (Tailwind CSS, FontAwesome)，這個 .exe 執行時電腦必須連上網路才能載入這些樣式。如果離線執行，畫面會變得很醜。

2. 可以改 App 圖示嗎？
可以。準備一個 .ico 格式的圖示檔 (例如 app.ico)，然後在打包指令中加入 --icon=app.ico：
pyinstaller --noconsole --onefile --icon=app.ico --add-data "github_pushV2.html;." launcher.py

3. 右鍵選單或複製貼上失效？
pywebview 預設可能沒有右鍵選單。您可以使用快捷鍵 Ctrl+C / Ctrl+V，這通常是支援的。

進階：製作完全離線版 (下載 CDN)

如果您希望程式在沒有網路的情況下也能正常顯示樣式（解決「介面看起來怪怪的」問題），您需要將線上的 CDN 資源下載下來。

1. 下載資源檔案

打開您的 HTML 檔，找到 <script src="https://..."> 或 <link href="https://..."> 的網址。

將網址貼到瀏覽器，按右鍵「另存新檔」，儲存在 MyWebApp 資料夾中。

例如：

Tailwind: 下載存成 tailwindcss.js

Marked: 下載存成 marked.min.js

(FontAwesome 和 Google Fonts 因為包含多個字體檔，手動下載較複雜，建議初學者先保留線上連結即可)

2. 修改 HTML 路徑
使用筆記本開啟 github_pushV2.html，將網址改成檔名：

<!-- 原本 -->
<script src="[https://cdn.tailwindcss.com](https://cdn.tailwindcss.com)"></script>

<!-- 改成 -->
<script src="tailwindcss.js"></script>


3. 修改打包指令
在打包時，您需要把這些新下載的檔案也告訴 pyinstaller。使用多個 --add-data 參數：

pyinstaller --noconsole --onefile --add-data "github_pushV2.html;." --add-data "tailwindcss.js;." --add-data "marked.min.js;." launcher.py


(注意：每個檔案都要用 --add-data "檔名;." 的格式加入)

常見問題 (HTML 特有)

1. 介面看起來怪怪的 / 樣式跑版？
這是因為電腦沒有連上網路，無法載入 CDN 樣式。請參考上方的「進階：製作完全離線版」章節來解決此問題。

2. 可以改 App 圖示嗎？
可以。準備一個 .ico 格式的圖示檔 (例如 app.ico)，然後在打包指令中加入 --icon=app.ico：
